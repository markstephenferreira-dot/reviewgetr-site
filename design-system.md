# Review Getr — Design System (UUPM-Derived)

## Style Direction
- **Primary Style:** Dark Mode with bold gold/yellow accents — premium, high-contrast, conversion-focused
- **Inspiration:** Comet Reviews — gold headlines on dark backgrounds, red CTAs, 3-step process
- **Landing Pattern:** Funnel (3-Step Conversion) — Hero > Problem > Solution > Action

## Color Palette
| Token | Hex | Usage |
|---|---|---|
| navy | #1E293B | Dark background sections |
| navy-deep | #0F172A | Deepest backgrounds |
| gold | #F59E0B | Headlines, star ratings, emphasis |
| cta-red / accent | #DC2626 | Primary CTA buttons |
| cta-red-hover | #B91C1C | CTA hover state |
| primary | #2563EB | Trust blue — links, secondary buttons |
| primary-dark | #1E40AF | Blue section backgrounds |
| text-dark | #0F172A | Body text on light backgrounds |
| text-light | #F8FAFC | Body text on dark backgrounds |
| white | #FFFFFF | Light section backgrounds |

## Typography
- **Headings:** Space Grotesk (bold, modern, techy)
- **Body:** DM Sans (clean, highly readable)
- **Scale:** H1 48-64px, H2 36-48px, H3 24-32px, Body 16-18px

## CTA Strategy
- Primary CTA: Red (#DC2626), full-width on mobile, bold uppercase text
- CTA Text: "START YOUR 14-DAY FREE TRIAL" / "BOOK YOUR FREE SETUP CALL"
- Repeated at: Hero, after how-it-works, after social proof, pricing, footer

## Section Pattern
- Dark sections: Navy background, light text, gold headings
- Light sections: White background, dark text
- Alternating dark/light creates visual rhythm

## Key Design Decisions
1. Red CTAs — higher urgency, proven conversion lift for trial/signup
2. Gold star motif — reinforces 5-star review theme throughout
3. Alternating backgrounds — clear visual rhythm, prevents fatigue
4. Single pricing plan — eliminates decision paralysis ($97/mo)
5. Competitor comparison — tangible proof of value (138 reviews vs 1-8)
6. Calendar embed placeholder — reduces friction for booking
